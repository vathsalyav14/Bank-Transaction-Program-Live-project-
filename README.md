# Bank-Transaction-Program-Live-project-
# A simple Python ATM transaction simulation. The program welcomes the user, asks them to insert their card, enter the withdrawal amount, and provide a PIN. The transaction details are then saved to a file. Built to practice basic Python concepts like user input and file handling.
#This is a small Python project where I created a simple ATM transaction simulation. The program interacts with the user just like a basic ATM would. It first welcomes the user, asks for their name, and then prompts them to insert their card. After that, the user can enter the amount they want to withdraw and their PIN to complete the transaction.

Once the transaction is completed, the program displays a confirmation message and saves the transaction details (name, amount, and PIN) into a text file.
I built this project to practice basic Python concepts like user input, program flow, and file handling. The goal was to keep the program simple and easy to understand while simulating a real-world ATM interaction.

Features
1. User welcome and interaction
2. ATM card insertion simulation
3. Withdrawal amount input
4. PIN entry
5. Transaction details saved to a file

Built With: Python
This project is part of my learning journey as I continue exploring programming and building simple real-world simulations.
